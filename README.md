# 🌩️ IDEAStorm

Bienvenue sur **IDEAStorm**, un jeu incrémental (Clicker) moderne et infini développé avec React.
Cliquez, automatisez et atteignez des scores astronomiques pour dominer le classement mondial !

## 🎮 JOUER MAINTENANT

Pas besoin d'installation ! Le jeu est accessible directement dans votre navigateur :

👉 **[CLIQUEZ ICI POUR JOUER](https://tobias-ferrand.ovh/ideastorm/index.html)** 👈

---

## ✨ Fonctionnalités

* **Progression Infinie :** Commencez par de simples clics et montez jusqu'aux rangs divins (**God Mode**) atteignant des nombres comme *100000 ZZ* !
* **La Boutique de Titouan :**
    * **Achats :** Améliorations de Clic, Gains Automatiques, et Multiplicateurs de puissance.
    * **Ventes :** Un système stratégique qui permet de revendre vos améliorations (remboursement de 50%) pour ajuster votre stratégie.
    * **Compagnons :** Collectionnez des alliés (Chats, Chats Ninja, Rois Chats, Volcan, et l'Oie).
* **Classement en Temps Réel (Leaderboard) :** Comparez votre score avec les autres joueurs grâce à une base de données connectée.
* **Sauvegarde Cloud :** Créez un compte pour sauvegarder votre progression automatiquement (Support des très grands nombres via Supabase).
* **Design Moderne :** Interface en **Glassmorphism** avec des arrière-plans thématiques.
* **Easter Egg :** Une fin secrète attend ceux qui atteindront les limites du jeu...

## 🛠️ Technologies

Ce projet utilise une stack technique moderne :

-   **Frontend :** React + Vite
-   **Gestion d'État :** Zustand (Gestion optimisée des nombres et de la boutique)
-   **Routing :** React Router (HashRouter pour compatibilité GitHub Pages)
-   **Backend & Base de données :** Supabase (PostgreSQL configuré en `NUMERIC` pour gérer les nombres infinis)
-   **Utilitaires :** Formatage personnalisé des grands nombres (k, M, b, a, aa... zz).
